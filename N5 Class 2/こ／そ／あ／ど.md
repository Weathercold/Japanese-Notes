# こ／そ／あ／ど

说话人和听话人相隔一段距离，**面对面**时：  
こ：距*说话人*较近的事物  
そ：距*听话人*较近的事物  
あ：距两人都较远的事物

说话人和听话人位于同一位置，**面向同一个方向**时：  
こ：距两人*较近*的事物  
そ：距两人*较远*的事物  
あ：距两人*更远*的事物

ど：疑问词

{こ、そ、あ、ど}+れ = 这个东西，那个东西，哪个东西［代词］  
{こ、そ、あ、ど}+の = 这个～，那个～，哪个～ ［限定词］  
{こ、そ、==あそ==、ど}+こ = 这里，那里，哪里［代词］  
{こ、そ、あ、ど}+ちら = 这边，那边，哪边［代词］

> [!example]  
> それはテレビです。：那是电视机。  
> あのノートは誰のですか。：那是谁的笔记本？  
> 李さんのパソコンはどれですか？：李先生的电脑是哪一台？
